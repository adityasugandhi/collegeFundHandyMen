# College Fund Handymen Website

A modern, responsive website for College Fund Handymen built with Vite and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
